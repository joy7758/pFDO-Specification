## Reporting a Vulnerability / 漏洞报告

- CN: 如发现安全漏洞，请优先通过 GitHub Security Advisory 或私信维护者，不要在公开 Issue 中披露可复现细节。
- EN: If you discover a vulnerability, report it via GitHub Security Advisory (preferred) or contact the maintainer privately. Do not disclose exploit details in a public issue.

Contact / 联系方式: `joy7759@gmail.com`

Response SLA / 响应时效:
- Acknowledgement / 初次确认: within 72 hours
- Initial triage / 初步分级: within 7 days
- Fix or mitigation plan / 修复或缓解方案: within 14 days
