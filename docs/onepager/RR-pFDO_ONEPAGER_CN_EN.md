## 1) What it is / 它是什么

**EN**  
**Physical FAIR Digital Object / 物理 FAIR 数字对象** is a RedRock Standard Domain identified by `RR-pFDO`.  
It defines *a stable governance surface* (IDs, specs, evidence, registry mapping) so implementations can evolve without losing traceability.

**中文**  
**Physical FAIR Digital Object / 物理 FAIR 数字对象** 是红岩宪章体系下的标准域 `RR-pFDO`。  
它定义一个稳定的“治理面”（编号、规范、证据、注册表映射），让实现可以迭代但不丢失可追溯性。

---

## 2) Why it matters / 为什么重要

**EN**  
- Reduce governance entropy: fewer surprises, fewer “unknown states”.  
- Make audits cheaper: evidence and traces become first-class outputs.  
- Enable adoption: plug-in integration without rewriting the world.

**中文**  
- 降低治理熵：减少系统“不可解释状态”。  
- 降低审计成本：证据与追踪成为一等公民。  
- 便于落地：可插拔接入，不要求推翻现有系统。

---

## 3) The promise / 我们交付的承诺

**EN**  
- A spec you can cite  
- An implementation you can run  
- Evidence you can reproduce

**中文**  
- 你能引用的标准文本  
- 你能跑起来的实现入口  
- 你能复现的证据与结果

## 4) Primary use-cases / 典型场景

**EN**  
- Make physical assets governable as executable digital objects
- Standard-first deployment: start from governance hub, then plug in data sources

**中文**  
- 把“物”变成可治理、可审计、可执行的数字对象
- 先标准后部署：先对齐宪章与标准域，再接入真实数据源
